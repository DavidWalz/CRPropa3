Detailed Description

Flags a particle when leaving the ellipsoid.

This module flags particles when outside of the ellipsoid, defined by two focal points and a major axis (length). The particle is made inactive and by default is flagged "OutOfBounds". Optionally the module can ensure the candidate does not overshoot the boundary by more than a set margin.
