void crpropa::loadGrid ( ref_ptr< VectorGrid >  grid,
std::string  filename,
double  conversion = 1 
)

Dump / load functions for scalar / 3-vector grids and binary / plain text files. The grid points are stored from (0, 0, 0) to (Nx, Ny, Nz) with the z-index changing the fastest. Vector components are stored per grid point in xyz-order.

In case of plain-text files the vector components are separated by a blank or tab and grid points are stored one per line. Also there can be any number of comment lines at the beginning of the file, started with a #.

All functions offer a conversion factor that is multiplied to all values.

Return the nucleus mass by lookup from a table. The masses are the atomic masses from the NIST database (http://www.nist.gov/pml/data/comp.cfm) minus electron masses, neglecting electron binding energies.
