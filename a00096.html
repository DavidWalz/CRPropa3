Detailed Description

Flag a particle when leaving the sphere.

This module flags particles when outside of the sphere, defined by a center and radius. The particle is made inactive and by default is flagged "OutOfBounds". Optionally the module can ensure the candidate does not overshoot the boundary by more than a set margin.
